# eval-to-json
Evaluate a JavaScript expression and print JSON output

## Installation

```bash
npm install --global eval-to-json
```

## Command-line usage

```bash
eval-to-json file.js
```

```bash
eval-to-json < file.js
```

```bash
cat file.js | require-to-json
```

```bash
echo '9 * 87 * 654 * 3210' | eval-to-json
```
